'[power]' - start new route/stop/resume current task
'[r]' - get next task
'[w]:start' - begin recording new route
'[w]:end' - end recording new route
'[w]:{"id":"string", "distance": "number", "degree": "number", "speed": "number", "timeout": "number"}' - string format, record new task to server

  distance: дистанция в км
  degree: угол в градусах;
  speed: скорость в км/ч;
  timeout: таймаут перед выполнением в мин;
